TERMOS DE USO
Descrição do Projeto
  • Finalidade: A Humanize-se é um projeto filantrópico voltado a atender as necessidades de portadores de necessidades especiais. Resume-se em permitir que donatários encontrem doadores dispostos a ceder o equipamento desejado. O projeto atua como intermediário entre doador e donatário, não interferindo no processo de comunicação ou entrega.
  • Parcerias: Empresas dispostas a realizar doação de lotes de equipamentos podem firmar parcerias com a Humanize-se. Anúncios especiais sobre a doação desses lotes serão exibidos na página inicial.
  • Restrição especial: A Humanize-se desencoraja ato de compra dos produtos. Nenhum anúncio deve conter cobranças de valores, o produto deve ser doado por livre e espontânea vontade.
1 PRODUTOS

1.1 A visualização dos produtos é livre para os visitantes, mas as informações de contato são restritas para usuários credenciados, para garantir a segurança do donatário;
1.2 Produtos podem ser removidos do sistema caso algum desses critérios sejam preenchidos:
1.2.1 Anexação de imagens ofensivas, pornográficas, comerciais ou não relacionadas com o anúncio;
1.2.2 Descrição do produto contendo palavras inapropriadas, propagandas indevidas/spamming e direcionamento para sites maliciosos;
1.2.3 Cobrança indevida;
1.3 Produtos podem ser removidos pelo próprio doador caso um donatário entre em contato e o processo de doação seja confirmado. Anúncios podem ter uma vida útil relativamente pequena;

2 CADASTRO

2.1 As informações fornecidas no cadastro devem ser verdadeiras, pois essas informações servirão como base de contato entre doador/donatário;
2.2 As informações fornecidas durante o cadastro serão visualizadas apenas por usuários credenciados (cadastrados);
2.3 O perfil pode ser removido do sistema caso algum desses critérios sejam preenchidos:
2.3.1 Pelo próprio usuário, caso ele não necessite mais utilizar este sistema;
2.3.2 Pelo administrador, caso o perfil se torne inativo por mais de 3 meses ou caso o usuário utilize o sistema de forma inadequada, contra o acordo aqui regido.

3 ANÚNCIO

3.1 O conteúdo dos anúncios deve ser íntegro e detalhado, contribuindo assim para a rapidez do fluxo contínuo de postagens e doações realizadas com sucesso;
3.2 Anúncios podem ser removidos do sistema caso algum desses critérios sejam preenchidos:
3.2.1 Pelo usuário, caso a doação seja finalizada ou quando não houver mais interesse pela doação;
3.2.2 Pelo administrador, caso o anúncio fique inativo ou fora das normas aqui descritas;
3.3 O anúncio não deve conter descrição ofensiva, enganosa ou maliciosa;
3.4 O anunciante não pode cobrar valores, bens ou serviços para realizar a doação;

4 DOAÇÃO

4.1 As informações de contato se encontram na página do anúncio;
4.2 O método de entrega do produto é escolhido pelos envolvidos (doador e donatário);
4.2.1 É preferível utilizar o serviço dos Correios para realizar as entregas, por questões de segurança;
4.2.2 Caso a doação seja feita pessoalmente, recomendam-se lugares seguros e movimentados;
4.2.3 A Humanize-se NÃO se responsabiliza pelo método de entrega adotado, atuando apenas como intermediária entre as partes;

5 SEGURANÇA

5.1 Informações pessoas são visíveis apenas para usuários credenciados;
5.2 Recomenda-se a criação e utilização de um e-mail secundário;
5.3 Recomenda-se o uso dos serviços dos Correios para as entregas;
5.4 Recomendam-se lugares movimentados para entregas pessoais;
5.5 Foto de perfil é opcional;
5.5.1 Imagens registradas ou comerciais não são permitidas;

Ao realizar o cadastro, o usuário estará de acordo com todos os termos aqui registrados, e compreende a importância do seguimento das normas, para garantir o bom funcionamento do sistema. O descumprimento das normas resultará na desativação permanente da conta. BÁRBARA PUTA
